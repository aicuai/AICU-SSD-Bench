Hier is een overzicht van de belangrijkste aspecten van **HTML** en **JavaScript**, gevolgd door een simpel **voorbeeld** van hoe ze samen kunnen worden gebruikt. --- ### 1. **HTML (HyperText Markup Language)** HTML is de basis van webpagina's. Het maakt de structuur van een pagina en bevat tags die het contenten begeleiden. **Voorbeeld van HTML-structuur:** ```html My First Web Page

Welkom op mijn website!

Je kunt hier je tekst plaatsen.

``` --- ### 2. **JavaScript (JS)** JavaScript is een scripting-taal die je kunt gebruiken om interactivity in webpagina's toe te voegen. Het werkt in het browserwindow en kan bijvoorbeeld tekst wijzigen of animaties maken. **Basisklassen van JavaScript:** - **Variables**: `let`, `const` - **Functions**: `function myFunction() { ... }` - **Events**: `onclick`, `onload` - **Arrays**: `let list = [1, 2, 3];` **Voorbeeld van een functie:** ```javascript function myFunction() { alert("Je hebt geklikt!"); } ``` --- ### 3. **CSS (Cascading Style Sheets)** CSS wordt gebruikt om de uitstraling van een pagina te bepalen. Het wordt meestal in een `.css`-bestand geplaatst en wordt via `` in HTML geïmporteerd. **Voorbeeld van CSS:** ```css body { font-family: Arial, sans-serif; background-color: #f0f0f0; } h1 { color: #333; } ``` --- ### 4. **Samenvatting van de drie technologieën** - **HTML** maakt de structuur van de pagina. - **JavaScript** maakt de interactivity. - **CSS** maakt de uitstraling van de pagina. --- ### 5. **Samenvattend voorbeeld: een interaktieve website** ```html Interaktieve Website

Welkom!

Je kunt hier je tekst plaatsen.

``` --- ### 6. **Tips** - Gebruik **``-taggen** om codeblokken te markeren (bijv. `function myFunction()`). - Gebruik een **code-editor** zoals VS Code of Sublime Text voor het schrijven van JavaScript en CSS. - Test je code in een browser (bijv. Google Chrome) om te zien hoe het werkt. --- Als je meer specifieke vragen hebt (bijv. hoe je een animatie maakt of een formulier toevoegt), laat het weten! 🌐💻
