Hier is een eenvoudige voorbeeld van een HTML-website met JavaScript en CSS. Dit laat zien hoe je een pagina kunt maken met HTML, JavaScript voor interactiviteit en CSS voor styling. ```html Basiskunst HTML + JavaScript

Welkom!

Hallo, wereld! Dit is een voorbeeld van een HTML + JavaScript website.

``` ### Wat het doet: 1. **HTML**: Maakt de structuur van de pagina (head en body). 2. **CSS**: Stylt de pagina (voorbeeld: achtergrond, lettertype, knop). 3. **JavaScript**: Maakt de interactiviteit (bijv. wisselt de tekst van een paragraaf wanneer een knop wordt geklikt). ### Hoe je het testt: 1. Sla het bestand op als `index.html`. 2. Open het in een webbrowser (zoals Chrome of Firefox). 3. Klik op de knop en zorg dat de tekst van de paragraaf wijkt. ### Extra tips: - Voeg meer elementen toe (zoals velden, afbeeldingen, animaties). - Gebruik `addEventListener` voor complexe interacties. - Gebruik `localStorage` of `fetch` voor meer functies. Als je meer specifieke functies wilt (zoals een formulier, een timer, of een animatie), laat het weten!
